# 文档说明

## 快速启动

- 安装依赖：

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

- 启动项目：

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
## 在线演示

访问线上演示环境：

https://stock-query-app.vercel.app/

## 项目说明

使用 FinMind 提供的股票资料 API ，实现台股数据查询功能。

项目使用 Next.js 14 框架开发，使用 MUI 组件库和 TailwindCSS 实现页面布局和图表展示，使用 Day.js 提供日期计算能力。
